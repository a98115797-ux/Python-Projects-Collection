# Python-Projects-Collection

> A professional, open-source Python curriculum. **50 projects**, each rebuilt at **three levels** (Beginner, Intermediate, Advanced) for **150 complete implementations**. Every variant ships runnable code, a GitHub-friendly Markdown guide, and a self-contained interactive HTML guide.

## Why this repo exists

The fastest way to *feel* your growth as a programmer is to build the same thing again, better. Here you build 50 real projects three times each:

- **Beginner** sees the idea (console, stdlib only, heavy comments).
- **Intermediate** sees the architecture (modules, validation, persistence).
- **Advanced** sees the engineering (OOP, type hints, tests, GUI/web).

Diff any project across levels and you are literally reading your own future skills.

## How to use it

1. Pick a project folder (start at `01-Calculator`).
2. Choose your level.
3. Read `GUIDE.md` (or open `guide.html` in a browser for the interactive version).
4. Run the code, do the practice challenges, take the quiz.
5. Move up a level, then on to the next project.

## The blueprint

The full specification (standards, themes, file rules, workflow, milestones) lives in [`MASTER_IMPLEMENTATION_GUIDE.md`](./MASTER_IMPLEMENTATION_GUIDE.md). It is the single source of truth for how every project is built.
