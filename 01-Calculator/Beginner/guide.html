Section 05

Key Concepts

1. Everything from input() is text

Typing 5 gives you the string "5", not the number 5. To do maths we convert with float().

2. Functions are reusable machines

A function takes something in, does a job, and hands something back. We build small functions like get_number() and calculate().

3. try / except stops crashes

If someone types "banana", float("banana") would crash. We wrap it in try/except to catch the mistake and ask again.

Section 08

Complete Source Code

This is the entire program. It matches the repository exactly.

main.py
"""Beginner Calculator.

Asks the user for two numbers and one operator (+, -, *, /),
then prints the result. Uses only Python's built-in features.
"""


def get_number(prompt):
    """Ask for a number, repeating until the user types a valid one."""
    while True:
        raw_value = input(prompt)
        try:
            return float(raw_value)
        except ValueError:
            print("  That doesn't look like a number. Try again.")


def get_operator():
    """Ask for one of the four supported operators."""
    valid_operators = ("+", "-", "*", "/")
    while True:
        operator = input("Choose an operator (+, -, *, /): ").strip()
        if operator in valid_operators:
            return operator
        print("  Please type one of: + - * /")


def calculate(first_number, operator, second_number):
    """Return the result of applying the operator to the two numbers."""
    if operator == "+":
        return first_number + second_number
    if operator == "-":
        return first_number - second_number
    if operator == "*":
        return first_number * second_number
    if operator == "/":
        if second_number == 0:
            return "Error: you cannot divide by zero."
        return first_number / second_number
    return "Error: unknown operator."


def main():
    """Run the calculator once, from start to finish."""
    print("=" * 40)
    print("        Welcome to the Calculator")
    print("=" * 40)

    first_number = get_number("Enter the first number:  ")
    operator = get_operator()
    second_number = get_number("Enter the second number: ")

    result = calculate(first_number, operator, second_number)

    print("-" * 40)
    print(f"Result: {first_number} {operator} {second_number} = {result}")
    print("-" * 40)


if __name__ == "__main__":
    main()

Section 09

Code Walkthrough

get_number() — safe input

The while True loop keeps asking until it succeeds. float(raw_value) tries to convert the text; if it fails, except ValueError runs and the loop goes again.

get_operator() — only valid choices

We store the four symbols in a tuple and only return when the input is one of them. .strip() removes stray spaces.

calculate() — the maths

A chain of if checks picks the operation. Before dividing, we check for zero and return a message.

main() and if __name__

main() wires everything together; the final if runs it only when the file is started directly.

Section 14

Deep Dive: why float and not int?

We use float() so users can enter decimals like 3.5. Floats are binary, which is why 0.1 + 0.2 equals 0.30000000000000004. For money you'd reach for decimal.Decimal. The Advanced variant explores precision further.
